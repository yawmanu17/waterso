Data Notes
How the app data flow works

• Customer monthly reads are the primary source of truth.

• Save & Next uses the active route order.

• Billing is recalculated from saved reads.

• NRW uses auto-summed customer reads unless manual override is selected.

• DMA geocode scope biases address suggestions to the selected service area.

• Printable report opens in a clean report view for PDF save / printing.
